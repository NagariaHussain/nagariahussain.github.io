Introduction

The main purpose of the website is to provide the user with Articles and Tutorials on the following topics:
  1. Technology and Programming
  2. Travel and Tourism
  3. Food
  4. News: Short Articles
I chose to implement it as 5 page website which has a home page and one page for each of the above topics. The goal was to create a responsive website that provides its user with information in an organized and beautiful manner.

3 ways in which my site is accessible

  1. The alt tags of all the image elements contain a description of the image to help users with disability to visualize the image.
  2. I have used proper semantic and element tags to make the different parts of the website more accessible. For instance, use of article and figure tags are used instead of plan div tags.
  3. Use of aria labels to make mobile navigation more accessible.
    "class="nav-button hamburger hamburger--elastic" aria-label="Menu" aria-controls="navigation" aria-expanded="false"

  2. CSS Custom Variables: When I wanted to theme my website, i.e., set a different theme color for each page using JavaScript, the requirement was to set a primary color and change it based on the URL the user is currently on. I came across the concept of CSS Custom Properties (a.k.a. CSS Variables). I learned it using this article on MDN. I have used 3 variables to control the color for a page:
    1. --primary-color
    2. --primary-backdrop-color
    3. --secondary-color
    I have used JavaScript to change the value associated with the variables based on the URL (relative path to be precise). For example, see the code snippet below:

    if (location.pathname === "/pages/shorts.html") {
    root.style.setProperty('--primary-color', "#ffc93c");
    root.style.setProperty('--primary-backdrop-color', "rgba(255, 201, 60, 0.6)");
    }
